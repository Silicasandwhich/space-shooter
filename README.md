# space shooter

Dependencies
-------------
- pyxel

    if you don't have pyxel, just run
    ``
    pip install pyxel
    ``
    as it has no dependencies

- sqlite3 (I'm on version 3.21.0 if it matters)




how to play
-------------

The game will tell you how to navigate windows

- shoot: space

- move: A and D
